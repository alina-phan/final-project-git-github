## Welcome
All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to Contribute

### 1) Report Bugs
- Use the Issues section to report bugs.
- Include clear steps to reproduce, expected behavior, and actual behavior.
- Add screenshots/logs if available.

### 2) Suggest Enhancements / Ideas
- Open an Issue describing your idea.
- Explain the problem it solves and any alternatives you considered.

### 3) Fix Bugs or Add Features
1. Fork the repository.
2. Create a new branch for your change:
   ```bash
   git checkout -b feature-or-fix-name
   ```
3. Make your changes.
4. Commit with a clear message:
  ```bash
   git commit -m "Describe your change"
   ```
5. Push to your fork:
  ```bash
   git push origin feature-or-fix-name
  ```
6. Open a Pull Request and describe what you changed and why.
